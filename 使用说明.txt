🚀 快速开始（3步）
================================================================================

第1步：安装依赖
--------------
  cd AI_Agent_Complete
  pip install -r requirements.txt

第2步：修改配置
--------------
  打开 config.yaml 文件，修改以下两行：
  
  sglang_path: "你的SGlang路径"     # 例如: D:/Code/sglang
  models_path: "你的模型路径"        # 例如: D:/Models

第3步：启动服务
--------------
  python start.py

然后浏览器打开：http://localhost:8000/chat

================================================================================
⚙️ 配置说明
================================================================================

必须配置的项：
--------------
1. sglang_path - SGlang代码目录路径
   示例: D:/Code/sglang 或 /home/user/sglang

2. models_path - 模型文件存放目录
   示例: D:/Models 或 /data/models

可选配置：
----------
- server.port: 修改服务端口（默认8000）
- model_mappings: 配置模型别名和路径
- sglang_service: 配置SGlang服务地址

================================================================================
🐛 常见问题
================================================================================

问题1：启动失败，提示找不到模块
  解决：pip install -r requirements.txt

问题2：无法访问前端页面
  解决：确认服务已启动，检查 http://localhost:8000/chat

问题3：端口被占用
  解决：修改 config.yaml 中的 port 参数

问题4：找不到模型
  解决：检查 config.yaml 中的 models_path 是否正确
